## 03:56-04:20 UTC - Google Cloud Billing Investigation & Solutions

### Issue Discovery
User reported **Rp 114K** Google Cloud billing for 28 days, tapi hanya:
- 14 images generated (expected ~Rp 9K)
- 4 videos generated (expected ~Rp 13K)
- **Total expected:** ~Rp 22K
- **Actual:** Rp 114K
- **Selisih:** ~Rp 92K tidak terjelaskan 😱

### Root Cause Analysis
Dari screenshot Google AI Studio + Cloud Billing:
1. **5M tokens Gemini 2.5 Flash** di Mar 19 (spike abnormal)
2. ~200-500 requests dalam satu hari
3. Kemungkinan: **Agent context bloat** (MEMORY.md, AGENTS.md terkirim setiap request)
4. **Automated loops/retries** yang tidak tercatat di file system
5. Failed requests tetap di-charge

### Solutions Implemented ✅

**SOLUSI 2: Limit Context Length**
- Created `generate-image-direct.sh` & `generate-video-direct.sh`
- Direct execution, bypass agent delegation → hindari context bloat
- Located: `/root/.openclaw/workspace/scripts/`

**SOLUSI 3: Add Retry Limit & Monitoring**
- Created `monitor-api-usage.sh` - log setiap image/video request
- Auto-detect rate spikes (>5 req/min → warning)
- Log file: `/root/.openclaw/workspace/logs/api-usage.log`
- Integrated into `generate-image.sh` & `generate-video.sh`

**SOLUSI 4: Monitor Logs untuk Detect Loops**
- Created `check-api-health.sh` - dashboard untuk audit usage
- Shows: request count, rate per hour, provider breakdown, alerts
- Usage: `check-api-health.sh [hours]`
- Example: `check-api-health.sh 24` (last 24 hours)

**SOLUSI 1: Console Billing Location (pending user)**

## 04:30 UTC - Google Veo Audio Charge Discovery 🚨

User menemukan charge **Rp 60,478** untuk "Veo 3 Fast Audio Generation (output)".

**Root Cause:**
Google Veo men-charge **VIDEO & AUDIO TERPISAH**:
- Video output: ~Rp 3-8K per video
- **Audio output: ~Rp 15K per video** 😱
- Total: ~Rp 18-23K per video (bukan Rp 3K seperti dikira!)

**Evidence:**
- 4 videos generated × Rp 15K audio = Rp 60K ✅
- Audio bukan dari TTS (generate-audio.sh), tapi dari video soundtrack

**Impact:**
- Previous cost estimation SALAH total
- Video generation jauh lebih mahal dari perkiraan
- Need update user expectation & limits

**Action Taken:**
- Updated TOOLS.md dengan cost breakdown accurate
- Recommend kie.ai Veo3 Fast (flat rate, audio included)
- Lower video limit: Max 2-3 videos/hari (bukan 3)

**New Video Cost Understanding:**
- Google Veo 3 Fast: Rp 18-23K/video (video + audio)
- Google Veo 3 Standard: Rp 40-50K/video
- kie.ai Veo3 Fast: ~Rp 10-15K/video (flat, audio included)
